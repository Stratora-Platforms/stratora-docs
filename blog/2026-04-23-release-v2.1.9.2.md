Stratora v2.1.9.2 is a maintenance release with a single fix for the Escalation Teams page. In-place upgrade from v2.1.9 or v2.1.9.1.

<!-- truncate -->

## Fixed

- **Escalation Teams page loads reliably on tenants with rotation teams** — The Escalation Teams page, alert configuration modal, and onboarding wizard's Alerts step now load correctly when any escalation team has a rotation schedule with a start date configured. Previous versions displayed a load error on these surfaces.

## Operator Notes

- In-place MSI upgrade from v2.1.9 or v2.1.9.1. No database migration or manual intervention required.
- No configuration changes. Existing escalation teams, rotation schedules, and on-call assignments are preserved as-is.

---

For installation and upgrade instructions, visit [docs.stratora.io](https://docs.stratora.io/docs/getting-started). The MSI is available on the [v2.1.9.2 release page](https://github.com/Stratora-Platforms/stratora-releases/releases/tag/v2.1.9.2).
