# Settings

The **Settings** page provides global configuration options that affect the Stratora platform.

Navigate to **Administration → Settings** to view and modify these options.

---

## Display Preferences

### Temperature Unit

Controls how temperature values are displayed across all dashboards, device detail pages, and monitoring panels.

| Option | Description |
|--------|-------------|
| **Celsius (°C)** | Default — temperatures shown in Celsius |
| **Fahrenheit (°F)** | Temperatures converted and shown in Fahrenheit |

This is a global setting that applies to all users. Changes take effect immediately across the platform.
